## 📌 What This Demonstrates

- Selecting DOM elements with `querySelector`
- Using a `for` loop to iterate over an array
- Dynamically creating `<li>` elements with `createElement`
- Appending elements to the DOM with `appendChild`
- Handling user interaction with `addEventListener`

## 🎯 Key Takeaways

- A `for` loop gives full control over the iteration — start index, end condition, and step — making it ideal when you need to track the index while looping
- Combining a `for` loop with `createElement` and `appendChild` is a common pattern for rendering dynamic lists
- Keeping DOM selection (`querySelector`) separate from the loop logic makes the code easier to read and maintain
