# ✨ JavaScript Rest & Spread Practice

A small practice project demonstrating the difference between the rest operator (collecting arguments into an array) and the spread operator (expanding an array into individual values) in JavaScript.

## 📌 What This Demonstrates

- Using the **rest operator** (`...numbers`) to collect any number of function arguments into an array
- Looping through the collected arguments to calculate a total
- Using the **spread operator** (`...students`) to expand an array's elements into a list
- Updating the DOM with `innerHTML` and template literals

## 🎯 Key Takeaways

- The rest operator (`...`) is used in a function's parameters to **gather** multiple arguments into a single array — useful when the number of inputs isn't fixed
- The spread operator (`...`) is used on an existing array or object to **expand** its values — commonly used for copying arrays, merging arrays, or passing array items as separate arguments
- Both use the same `...` syntax, but the context (function parameters vs. array/object literal) decides whether it's "rest" or "spread"
