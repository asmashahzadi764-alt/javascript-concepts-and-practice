## 📌 What This Demonstrates

- Selecting DOM elements with `querySelector`
- Using a `while` loop to repeat an action until a condition is met
- Clearing previous DOM content with `innerHTML = ""`
- Dynamically creating `<li>` elements with `createElement`
- Appending elements to the DOM with `appendChild`

---

## 🎯 Key Takeaways

- A `while` loop is best when the number of iterations depends on a condition rather than a fixed count
- Clearing the container (`innerHTML = ""`) before re-rendering prevents duplicate list items on repeated clicks
- Always update the loop variable (`number++`) inside the loop to avoid creating an infinite loop
