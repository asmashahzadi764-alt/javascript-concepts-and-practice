# 🔀 JavaScript Switch Day Finder

A small practice project demonstrating how to use a `switch` statement together with user input to find and display the day of the week matching a number entered by the user.

## 📌 What This Demonstrates

- Reading user input from a number field with `.value`
- Converting input string values to numbers with `Number()`
- Using a `switch` statement to match input against multiple `case`s
- Using `break` to stop execution once a matching case is found
- Handling invalid or out-of-range input with a `default` case
- Updating the DOM with `textContent`

---

## 🎯 Key Takeaways

- Input field values are always strings by default, so `Number()` is needed before using them in numeric comparisons like `switch` cases
- A `default` case is essential when working with user input, since values outside the expected range must still be handled gracefully
- Combining `switch` with real user input (instead of a hardcoded value) makes the logic feel closer to a real-world use case
