#  JavaScript Ternary Operator Practice

A small practice project demonstrating how to use the ternary operator (`? :`) in JavaScript as a shorthand for `if...else` to check eligibility based on user input.

## 📌 What This Demonstrates

- Reading user input from a number field with `.value`
- Converting input string values to numbers with `Number()`
- Using the ternary operator (`condition ? valueIfTrue : valueIfFalse`) as a shorthand conditional
- Updating the DOM with `textContent`

## 🎯 Key Takeaways

- The ternary operator is a compact one-line alternative to a simple `if...else` block, best used for short, single conditions
- Overusing or nesting ternaries can hurt readability — for complex logic, a regular `if...else` or `switch` is clearer
- As with any numeric comparison on input values, `Number()` conversion is needed before evaluating the condition
